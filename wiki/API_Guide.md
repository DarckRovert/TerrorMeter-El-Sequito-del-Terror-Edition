# API Guide — TerrorMeter Sequito 🛠️

### TerrorMeter:GetUnitThreat(unit, target)
Retorna el porcentaje de amenaza de una unidad con respecto al tanque actual en el objetivo.

### TerrorMeter:ForceReset()
Limpia todas las tablas de amenaza y fuerza un nuevo escaneo de raid.
