# Comandos de TerrorMeter v9.3.0

| Comando | Alias | Descripción |
|---|---|---|
| /terrormeter | /tm | Abrir/cerrar ventana principal |
| /tm reset | — | Resetear datos del encuentro actual |
| /tm clear | — | Limpiar historial completo |
| /tm modo dps | — | Cambiar a vista de DPS |
| /tm modo hps | — | Cambiar a vista de HPS |
| /tm modo threat | — | Cambiar a vista de Amenaza |
| /tm report | — | Publicar resultados en el chat |
| /tm window <n> | — | Ventana de tiempo (5, 15, 60, sesion) |
| /tm lock | — | Bloquear/desbloquear posición de ventana |
| /tm alpha <0-1> | — | Ajustar transparencia |
| /tm scale <n> | — | Ajustar tamaño (0.5 – 2.0) |
| /tm config | — | Abrir menú de configuración |
