## 📊 ¿Qué es TerrorMeter?

TerrorMeter es un medidor de daño y sanación de alto rendimiento, diseñado exclusivamente para el ecosistema de **El Séquito del Terror**. Analiza el combate en tiempo real y publica sus métricas directamente en el WCS_Brain.

## 🚀 Características

| Módulo | Descripción |
|---|---|
| **DPS en Tiempo Real** | Daño por segundo con ventanas deslizantes (5s, 15s, 60s, sesión) |
| **HPS Integrado** | Sanación por segundo con desagregado por tipo de hechizo |
| **Threat Display** | Indicador visual de amenaza con umbral configurable |
| **Multi-Segmento** | Separa automáticamente los encuentros de combate |
| **Brain Bridge** | Publicación de métricas directamente en el panel WCS_Brain |
| **Sync P2P** | Comparte métricas con miembros del clan vía addon channel |

## ⚙️ Instalación

1. Extrae en Interface\AddOns\TerrorMeter\.
2. Activa el addon.
3. Usa /terrormeter o /tm para abrir el panel.

## 💻 Comandos

| Comando | Descripción |
|---|---|
| /tm | Abrir/cerrar ventana |
| /tm reset | Resetear datos del encuentro actual |
| /tm clear | Limpiar todo el historial |
| /tm modo <dps/hps> | Cambiar vista |

Ver [COMMANDS.md](./COMMANDS.md) para la lista completa.
